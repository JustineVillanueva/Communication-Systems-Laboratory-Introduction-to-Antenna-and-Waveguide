# Communication Systems Laboratory: Introduction to Antenna and Waveguide
### Objective
- To understand the basic principles and functions of antennas and waveguides.
- To learn about their functions and main components
- To understand how these trainers demonstrate fundamental concepts in antennas and waveguides.

---
### Introduction to the Antenna Trainer
The Scientech 2261 Antenna Training System is an educational kit designed for students to learn the basics of antennas. It provides hands-on understanding of how antennas transmit and receive electromagnetic waves, and allows visualization of concepts such as radiation patterns, gain, polarization, and beamwidth.
